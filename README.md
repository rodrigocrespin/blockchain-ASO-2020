# Blockchain concept (ASO 5K4 2020)

# Backend

## Dependencies:

- .NET Core 3.1 SDK

## How to run test console
```
cd Blockchain.TestConsole
dotnet run Blockchain.TestConsole.csproj
```

## How to run uni-tests

```
cd Blockchain.Tests
dotnet test
```


# Frontend

## Dependencies:

- Node JS v12.16.3


## How to run

```
npm install
npm start
```
